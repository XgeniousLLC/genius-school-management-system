# Hostel

Go to **Facilities → Hostel** to manage hostel blocks, rooms, and student allocations.

---

## Adding a Hostel Block

1. Go to **Hostel → Blocks**.
2. Click **Add Block** (e.g. Block A – Boys, Block B – Girls).
3. Enter the block name and warden name.
4. Click **Save**.

---

## Adding Rooms

1. Go to **Hostel → Rooms**.
2. Click **Add Room**.
3. Select the block, enter the room number, and set the capacity.
4. Click **Save**.

---

## Allocating a Student to a Room

1. Go to **Hostel → Allocations**.
2. Click **Allocate Student**.
3. Select the student and the room.
4. Set the check-in date.
5. Click **Save**.

The room's occupancy count updates automatically. Rooms that are full are marked as unavailable.

---

## Checking a Student Out

1. Find the allocation record.
2. Click **Check Out**.
3. Confirm the check-out date.
