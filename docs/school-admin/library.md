# Library

Go to **Facilities → Library** to manage books and track issues and returns.

---

## Adding a Book

1. Go to **Library → Books**.
2. Click **Add Book**.
3. Enter:
   - Title and Author
   - ISBN (optional)
   - Category (e.g. Science, Fiction, Reference)
   - Number of copies available
   - Shelf/rack location (optional)
4. Click **Save**.

---

## Issuing a Book to a Student

1. Go to **Library → Issue Book**.
2. Search for and select the **Student**.
3. Search for and select the **Book**.
4. Set the **Due Date** for return.
5. Click **Issue**.

The book's available copies decrease by one automatically.

---

## Returning a Book

1. Go to **Library → Issued Books**.
2. Find the record (search by student name or book title).
3. Click **Return**.
4. If the book is overdue, the system calculates the **fine** automatically.
5. Record any fine collected and click **Confirm Return**.

---

## Searching for a Book

Use the search box on the Books page to find any book by title, author, or ISBN. The availability (how many copies are in/out) is shown next to each book.
