# Initial School Setup

Before adding students or staff, set up the building blocks of your school.

---

## School Profile

Go to **Admin → Settings** to fill in your school's basic details.

1. Enter the **School Name**, email, phone, address, and website.
2. Upload your **School Logo** (shown in the sidebar and on reports).
3. Upload a **Favicon** (the small icon in the browser tab).
4. Set your **Timezone**, **Currency**, and **Language**.
5. Click **Save General Settings**.

---

## Academic Year

An academic year must exist before you can enrol students or mark attendance.

1. Go to **School Setup → Academic Years** (or it may be auto-created during installation).
2. Make sure one year has **Is Current** turned on — this is the active year used across the system.

---

## Classes and Sections {#classes-and-sections}

**Classes** represent grade levels (e.g. Class 1, Class 2, Grade 10).
**Sections** are subdivisions within a class (e.g. Section A, Section B).

### Add a Class

1. Go to **School Setup → Classes**.
2. Click **Add Class**.
3. Enter the class name (e.g. `Class 5`) and a numeric order (e.g. `5` for sorting).
4. Click **Save**.

### Add a Section

1. Go to **School Setup → Sections**.
2. Click **Add Section**.
3. Select the **Class** this section belongs to.
4. Enter the section name (e.g. `A`, `Blue`, `Morning`).
5. Click **Save**.

> You can add multiple sections per class. Repeat for each class you need.

---

## Subjects {#subjects}

1. Go to **School Setup → Subjects**.
2. Click **Add Subject**.
3. Enter the subject name (e.g. `Mathematics`, `English`).
4. Optionally assign it to specific classes.
5. Click **Save**.

---

## Shifts

Shifts let you manage schools that run in multiple time slots (morning, day, evening).

1. Go to **School Setup → Shifts**.
2. Click **Add Shift**.
3. Enter the shift name and start/end times.
4. Click **Save**.

---

## Holidays

Mark holidays so they are excluded from attendance calculations.

1. Go to **School Setup → Holidays**.
2. Click **Add Holiday**.
3. Enter the holiday name, date (or date range), and an optional description.
4. Click **Save**.

> Holidays appear in the attendance calendar so staff know which days to skip.
