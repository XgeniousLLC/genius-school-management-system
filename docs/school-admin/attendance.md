# Attendance

Go to **Academic → Attendance** to mark and review student attendance.

---

## Marking Attendance

1. Go to **Academic → Attendance**.
2. Select the **Class**, **Section**, and **Date**.
3. Click **Load Students** — the list of students in that class appears.
4. For each student, select their status:
   - ✅ **Present**
   - ❌ **Absent**
   - 🕐 **Late**
   - 📋 **Excused**
5. Add a **Remark** for any student if needed (e.g. "Doctor's appointment").
6. Click **Submit Attendance**.

> Attendance can only be marked once per class per date. If you need to correct a mistake, edit the existing record.

---

## Editing Attendance

1. Go to **Academic → Attendance**.
2. Select the class, section, and the date you want to correct.
3. The already-saved statuses will be pre-selected.
4. Make your changes and click **Update Attendance**.

---

## Viewing the Attendance Report

For a summary view go to **Reports → Attendance**. You can filter by:

- Class
- Date range
- Status (present / absent / late / excused)

The report shows each record with the student name, date, and status.

---

## Attendance in Student Profiles

When you open a student's profile you will see their attendance percentage for the current month and a colour-coded calendar:

- 🟢 Green = Present
- 🔴 Red = Absent
- 🟡 Yellow = Late
