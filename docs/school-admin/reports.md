# Reports

Go to **Reports** in the sidebar to access all reporting tools.

---

## Dashboard Report

The **Reports → Dashboard** gives a live overview of your school:

- Total students, staff, attendance percentage, fees collected this month
- Outstanding fee balance
- Fee collection chart (last 6 months)
- Attendance trend chart (last 7 days)
- Recent activity log

---

## Attendance Report

**Reports → Attendance** shows individual attendance records.

**How to use:**

1. Select a **Class** (optional — leave blank for all classes).
2. Select a **Date Range**.
3. Filter by **Status** (Present, Absent, Late, Excused) — optional.
4. Click **Apply Filter**.

The table shows each record with the student name, date, class, and status.

---

## Academic Report

**Reports → Academic** shows exam performance by class.

**How to use:**

1. Select an **Exam** from the dropdown.
2. The report shows each class with:
   - Average marks percentage
   - Pass rate
3. Click on a class to drill down into individual subject results.

---

## Finance Report

**Reports → Finance** shows fee collection data.

**How to use:**

1. Select a **Date Range**.
2. The report shows:
   - Total collected in the period
   - Total outstanding balance
   - This month's payroll total
   - A daily collection chart
   - A detailed list of all payments with student names

---

## Custom Report

**Reports → Custom Report** lets you build your own report by selecting:

- Report type (Students, Attendance, Marks)
- Class and section filters
- Date range
- Which columns to include

Click **Generate Report** then **Export** to download as Excel or PDF.

---

## Audit Log

**Reports → Audit Log** records every action taken in the system — who did what and when.

Use this to:
- Investigate unexpected changes
- Review user activity
- Ensure data integrity

You can filter by user, action type, and date range.
