# Teacher Guide

As a teacher, you can mark attendance, assign homework, enter exam marks, and communicate with students and parents.

## Your Sidebar Menu

After logging in you will see these sections:

| Section | What You Can Do |
|---|---|
| **Dashboard** | Overview of your classes, attendance, upcoming exams |
| **Students** | View students in your classes |
| **Timetable** | See your teaching schedule |
| **Attendance** | Mark attendance for your classes |
| **Examinations** | Enter exam marks |
| **Homework** | Assign and manage homework |
| **Lesson Plans** | Create lesson plans |
| **Syllabus** | Track syllabus coverage |
| **Online Classes** | Add online class links |
| **Announcements** | Post class announcements |
| **Messages** | Send and receive messages |
| **Notifications** | View system notifications |
