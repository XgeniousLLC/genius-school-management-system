# Entering Exam Marks

1. Go to **Academic → Examinations**.
2. Find the exam you need to enter marks for and click **Enter Marks**.
3. Select the **Subject**.
4. For each student in the list:
   - Enter **Marks Obtained**
   - The **Total Marks** field should already be set
   - Tick **Absent** if the student did not attend
5. Click **Save Marks**.

Repeat for each subject you teach.

> Grades are calculated automatically — you do not need to enter them.

## Tips

- You can come back and edit marks before results are published.
- Once the admin publishes results, students and parents can see them in their portals.
- If a student's name is missing from the list, contact the school admin — the student may not be enrolled in the correct class.
