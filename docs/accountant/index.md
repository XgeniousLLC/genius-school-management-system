# Accountant Guide

As an accountant, you manage fee collection, fee structures, payroll, and financial reports.

## Your Sidebar Menu

| Section | What You Can Do |
|---|---|
| **Dashboard** | Overview of collection, outstanding fees |
| **Fee Payments** | Record and view fee payments |
| **Fee Structures** | Define how much each class pays |
| **Fee Categories** | Organise fees by type |
| **Payroll** | Process monthly staff salaries |
| **Finance Report** | Detailed financial summaries |
