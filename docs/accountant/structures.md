# Fee Structures & Categories

## Fee Categories

Categories group similar types of fees. Examples: Tuition Fee, Exam Fee, Transport Fee, Library Fee.

1. Go to **Finance → Fee Categories**.
2. Click **Add Category**, enter the name, and save.

## Fee Structures

Structures define the exact amount each class pays for each category.

1. Go to **Finance → Fee Structures**.
2. Click **Add Structure**.
3. Select a **Category**, **Class**, and enter the **Amount**.
4. Set the frequency — Monthly, Quarterly, Annually, or One-time.
5. Click **Save**.

> If different classes pay different amounts for the same fee (e.g. tuition is higher for Class 10 than Class 1), create a separate structure for each class.
